Vom 2D zum 3D: Extrude, Revolve, Press-Pull

⏱ 45 Minuten 🎯 Einsteiger 📦 Erste echte 3D-Körper

Lernziele

  • Ich kann eine geschlossene Skizze mit Extrude in einen 3D-Körper überführen.
  • Ich nutze Revolve, um rotationssymmetrische Teile (Vase, Schraube, Knauf) zu erzeugen.
  • Ich verwende die Modi New Body, Join, Cut, Intersect, New Component bewusst.
  • Ich kann mit Press-Pull bestehende Geometrie schnell verändern.
  • Ich erstelle Bohrungen mit Cut-Extrude oder dem speziellen Hole-Werkzeug.

Vorwissen-Check

Welche Operation eignet sich für eine Schraube oder eine Vase?
Revolve – beide Teile sind rotationssymmetrisch um eine Achse.
Was muss eine Skizze haben, damit Extrude einen Körper erzeugt?
Sie muss ein geschlossenes Profil haben (eine Fläche, die Fusion automatisch erkennt und beim Hovern hellblau einfärbt).

Theorie kompakt

Extrude (E)

Zieht ein 2D-Profil in die dritte Dimension. Die wichtigsten Optionen im Dialog:

  • Direction: One Side, Two Sides oder Symmetric (wenn die Skizze eine Mittelebene markieren soll).
  • Extent: Distance (mm), To Object (bis zu einer Fläche), All (durch alles).
  • Operation: New Body / Join / Cut / Intersect / New Component – darüber gleich mehr.
  • Taper Angle: Schräge beim Extrudieren, hilfreich z. B. für Entformungsschrägen.

Operationsmodi – das ist Gold wert

New BodyErzeugt einen separaten Körper. Empfohlen, wenn das Teil nicht zu einem bestehenden gehört.
JoinAddiert das neue Volumen zum bestehenden Körper.
CutSubtrahiert. Klassiker für Bohrungen und Aussparungen.
IntersectBehält nur den Schnittbereich – z. B. für komplexe Übergänge.
New ComponentWie New Body, aber als eigenständige Komponente. Wichtig für Baugruppen (Modul 5).
Faustregel Wenn du noch unschlüssig bist: New Component wählen. Aus Komponenten lassen sich später Bodies extrahieren, umgekehrt ist das deutlich aufwendiger.

Revolve (R ist im Sketch belegt → über Toolbar)

Rotiert ein Profil um eine Achse. Achse kann eine Origin-Achse, eine Sketch-Linie oder eine Construction Line sein. Voller Umlauf 360°, oder beliebiger Winkel. Wichtig: das Profil darf die Achse nicht überschneiden, sonst entstehen ungültige Geometrien.

Press-Pull (Q)

Magisches Multifunktionstool. Was es macht, hängt davon ab, was du auswählst:

  • Eine Skizzenfläche ausgewählt → wirkt wie Extrude.
  • Eine Bauteilfläche ausgewählt → verschiebt die Fläche, der Körper wird länger/kürzer.
  • Eine Kante ausgewählt → Fillet/Chamfer-Vorschau.

Schnell und intuitiv für Korrekturen, kann aber die Timeline aufblähen, wenn man es übertreibt.

Hole-Werkzeug

Für genormte Bohrungen (Durchgangsloch, Senkkopf, Sackloch, Gewindebohrung) gibt es ein eigenes Hole-Tool. Es ist „intelligent": Position kann an Skizzenpunkten oder Flächenmittelpunkten andocken, Metrische Schraubgrößen sind hinterlegt. Wir nutzen es ab Modul 3 intensiv.

Geführte Übung – L-Halterung 3D

Aus dem L-Profil aus Lektion 3 wird jetzt ein echtes Bauteil.

  1. Datei öffnen. Im Data Panel Lektion-3-LProfil öffnen. Speichere sofort als Lektion-4-LHalterung (Datei → Save As, oder Save Version dann umbenennen).
  2. Skizze sichtbar machen. Im Browser unter Sketches die Skizze sichtbar schalten. Hover mit der Maus über die L-Form: die geschlossene Fläche wird hellblau.
  3. Extrudieren. E drücken. Klicke auf die L-Fläche (nicht den Kreis – der Kreis kommt gleich). Im Dialog: Direction = One Side, Distance = 5 mm, Operation = New Body. OK.
  4. Bohrung als Cut. Skizze ist immer noch da. E erneut. Wähle den Kreis. Distance = -5 mm (negativ = nach unten durch das Bauteil), Operation = Cut. OK.
  5. Skizze ausblenden. Browser → Sketches → Auge aus. Du siehst jetzt nur den Körper mit der Bohrung.
  6. Rückseite verstärken. Klicke die Rückfläche an, Create Sketch (Fusion legt automatisch eine neue Skizze auf dieser Fläche an). Zeichne ein Rechteck, das die Rückseite vollständig überdeckt (Konturen einrasten lassen). Skizze schließen, E, Distance = 10 mm, Operation = Join. So entsteht eine Verstärkungsplatte am Rücken.
  7. Press-Pull testen. Q, klicke die Vorderfläche der Halterung an, ziehe leicht nach vorn. Distanz 2 mm eingeben, OK. Die ganze Vorderfläche verschiebt sich – der Körper ist 2 mm dicker geworden.
  8. Speichern. Strg/⌘+S. Versionskommentar: „Erste 3D-Halterung".

Bonus-Übung: Revolve einer Vase

  1. Neues Design. Lektion-4-Vase.
  2. Skizze auf YZ-Ebene. Damit die Vase senkrecht „steht".
  3. Mittellinie als Construction Line. Linie senkrecht entlang der Z-Achse zeichnen, mit X in eine Construction Line wandeln. Sie wird gestrichelt.
  4. Profil zeichnen. Mit L ein geschlossenes Profil rechts der Construction Line – geschwungener Hals, breiter Bauch, schmaler Boden. Konkrete Maße sind frei wählbar (z. B. 80 hoch, 35 max breit).
  5. Profil schließen. Wieder zur Construction Line zurück, alle Endpunkte mit Coincident verbinden. Skizze beenden.
  6. Revolve. Toolbar Create → Revolve. Profil wählen, dann die Construction Line als Axis. Angle = 360°. OK.
  7. Hohl machen (Vorgriff auf Shell). Toolbar Modify → Shell. Obere Fläche auswählen, Wandstärke 2 mm. OK.
  8. Speichern.
Erwartetes Ergebnis Eine 3D-L-Halterung mit Bohrung und Rückenverstärkung sowie eine erste Revolve-Vase mit Wand. Beide gespeichert.

Freie Transferaufgabe – Schraubenkopf

Konstruiere einen einfachen Sechskant-Schraubenkopf ohne Gewinde:

  • Sechskant mit Schlüsselweite (SW) 10 mm, Höhe 4 mm – via Polygon (inscribed) in einer Skizze, dann Extrude.
  • Schaft ⌀ 6 mm, Länge 25 mm – über zweite Skizze auf der Unterseite, Extrude Join.
  • Zentrale Bohrung im Kopf ⌀ 3 mm, 2 mm tief – Cut.

Akzeptanzkriterien: Bauteil heißt Lektion-4-Schraubenkopf, Maße per User-Parameter ausgelagert (vorgreifend; reicht es, wenn du sie als normale Maße einträgst – Parameter folgen in Modul 2).

Selbstkontroll-Quiz

1. Welche Operation eignet sich für eine Schraube?
Revolve. Schaft und Kopf sind rotationssymmetrisch.
2. Was bedeutet Cut bei Extrude?
Das neue Volumen wird vom bestehenden Körper subtrahiert. So entstehen Bohrungen oder Taschen.
3. Wofür sind Construction Lines gut?
Sie helfen beim Konstruieren (z. B. als Drehachse oder Symmetrie-Linie), zählen aber nicht zum Profil und werden ignoriert, wenn Fusion eine geschlossene Fläche sucht.
4. Was passiert, wenn du Press-Pull auf einer Bauteilfläche anwendest?
Die Fläche wandert in ihre Normalrichtung – der Körper wird länger oder kürzer in dieser Richtung.
5. Was bedeutet Symmetric bei Extrude-Direction?
Die Extrusion geht zu beiden Seiten der Skizzenebene gleich weit. Praktisch, wenn die Skizzenebene die Mittelebene des Bauteils sein soll.

Häufige Stolperer

  • „Extrude erkennt kein Profil" → die Skizze ist nicht geschlossen. Mit Trim oder zusätzlichen Linien schließen.
  • „Cut macht ungewollt einen Body weg" → falsches Target ausgewählt. Im Extrude-Dialog unter Objects to Cut manuell festlegen.
  • „Revolve gibt Fehler" → Profil schneidet die Achse oder Achse stimmt nicht. Construction Line muss außerhalb oder genau auf einer Profilkante liegen.
  • „Press-Pull will nicht" → bei sehr komplexer Geometrie zickig. Dann lieber explizit Extrude oder Move/Copy nutzen.
