Patterns: rechteckig, kreisförmig, Spiegeln, Pfad

⏱ 40 Minuten 🎯 Vertiefung ♻ Wiederholungen

Lernziele

  • Ich erstelle Rectangular Patterns mit Anzahl und Abstand pro Achse.
  • Ich nutze Circular Patterns um eine Achse, vollständig oder teilweise.
  • Ich spiegle Geometrie mit Mirror an einer Ebene.
  • Ich verwende Path Patterns entlang einer Kurve.
  • Ich pattern auf Bodies, Faces, Sketches und Features – und weiß, wann was sinnvoll ist.
  • Ich kann einzelne Pattern-Instanzen unterdrücken.

Vorwissen-Check

Worauf kannst du Patterns anwenden?
Auf Features (Operationen aus der Timeline), Bodies (ganze Körper), Faces (Flächen) und Components (eigenständige Bauteile, Modul 5).
Warum ist Spiegeln oft besser als zweimal das gleiche zeichnen?
Mirror erzeugt eine echte parametrische Verknüpfung: Änderst du das Original, ändert sich der Spiegel mit. Bei zwei separaten Skizzen müsstest du beide pflegen.

Theorie kompakt

Rectangular Pattern

Wiederholt ein Element entlang einer oder zwei Richtungen. Du brauchst:

  • Pattern Type: Faces / Features / Bodies / Components.
  • Objects: was patterniert wird.
  • Direction: eine Kante, Achse oder Construction Axis als Richtungsvektor.
  • Quantity / Distance: Anzahl und Gesamtabstand (oder Spacing pro Instanz).

Circular Pattern

Rotiert Elemente um eine Achse. Wichtige Optionen:

  • Axis: meist eine Construction Axis durch ein Bohrungszentrum oder eine Origin-Achse.
  • Total Angle: 360° = vollständig herum, sonst Teilkreis.
  • Compute Option: Optimized (schneller), Identical (sicherer), Adjust (für komplexe Fälle).

Mirror

Spiegelt Geometrie an einer Ebene. Mirror geht oft über Construction-Ebenen oder Origin-Ebenen. Auch Sketches lassen sich innerhalb einer Skizze spiegeln (Sketch-Toolbar → Mirror).

Path Pattern

Verteilt Elemente entlang einer Kurve – Spline, Bogen oder Sketch-Linie. Quantity wählst du explizit, Spacing kann Distance (gleiche Abstände) oder Spacing (gleichmäßig über die Kurve verteilt) sein. Sehr nützlich für Reihen entlang nicht-geradliniger Konturen.

Suppress / Skip Instances

Im Pattern-Dialog gibt es einen Bereich Suppress. Klicke einzelne Vorschau-Punkte an, die nicht erstellt werden sollen. So entstehen Lochbilder mit gewollten Lücken, ohne das Pattern zu zerstören.

Was patterniere ich womit?

Bohrung in Reihe entlang einer KanteRectangular Pattern, Type Features
Speichen eines RadesCircular Pattern, Type Bodies oder Features
Symmetrische zweite BauteilhälfteMirror, Type Bodies, Plane = Origin
Treppenstufen entlang eines SplinesPath Pattern, Type Bodies

Profi-Hinweis Patterniere immer das kleinste mögliche Element. Patternst du den ganzen Body, ist die Operation teurer als wenn du nur das eine Cut-Feature pattern­st.

Geführte Übung – gelochtes Lüftungsgitter

  1. Neues Design. Lektion-2-4-Lueftungsgitter.
  2. Parameter anlegen. 
    breite           = 100 mm
hoehe            = 60 mm
dicke            = 3 mm
loch_d           = 4 mm
loch_abstand     = 10 mm
loch_anzahl_x    = floor((breite - 20 mm) / loch_abstand)
loch_anzahl_y    = floor((hoehe  - 20 mm) / loch_abstand)
    Das ist clever: floor sorgt dafür, dass beim Verbreitern nie mehr Löcher angelegt werden, als hineinpassen.
  3. Grundplatte. Skizze auf XY: Rechteck zentrisch (Symmetric um beide Origin-Achsen). Maße = breite und hoehe. Skizze schließen, extrudieren symmetrisch um dicke / 2.
  4. Erste Bohrung. Skizze auf der Oberseite. Kreis vom Origin verschieben: Position auf -(breite/2) + 10 mm in X und -(hoehe/2) + 10 mm in Y, Durchmesser = loch_d. Skizze schließen. E, Profile = Kreis, Distance = -dicke, Operation = Cut.
  5. Rectangular Pattern. Create → Pattern → Rectangular Pattern.
    • Pattern Type: Features.
    • Objects: das Cut-Extrude in der Timeline.
    • Direction One: X-Achse aus dem Origin (Browser → Origin → X Axis).
    • Quantity One: loch_anzahl_x.
    • Distance One: loch_abstand * (loch_anzahl_x - 1).
    • Direction Two: Y-Achse.
    • Quantity Two: loch_anzahl_y.
    • Distance Two: loch_abstand * (loch_anzahl_y - 1).
    OK. Du hast jetzt ein gleichmäßiges Lochmuster.
    Variante Alternativ Spacing = Spacing wählen, dann gibst du den Abstand zwischen zwei Instanzen direkt an. loch_abstand kannst du dann ohne Multiplikation einsetzen.
  6. Stresstest. Modify → Change Parameters: breite auf 200 mm, loch_anzahl_x wird automatisch 18. Pattern füllt sich neu. Wenn die Distance-Formel nicht passt: nochmal nachprüfen.

Bonus-Übung – Speichenrad mit Mirror und Circular

  1. Neues Design. Lektion-2-4-Speichenrad.
  2. Felgenring. Zwei konzentrische Kreise auf XY (innen ⌀60, außen ⌀80). Skizze schließen, Profil = Ring extrudieren 8 mm. New Body.
  3. Eine Speiche zeichnen. Skizze auf XY: schmales Rechteck 4×30, vom Origin nach oben in den Ring hinein. Symmetric um Y-Achse, an Ring-Innenseite koinzident. Skizze schließen, extrudieren 8 mm Join.
  4. Circular Pattern. Create → Pattern → Circular Pattern. Pattern Type: Features. Objects: das Speichen-Extrude. Axis: Z-Achse (Origin). Quantity: 6. Total Angle: 360°. OK.
  5. Mirror als Aufgabe. Versuche dasselbe Muster zusätzlich an der XY-Ebene zu spiegeln (also Speichen oben + unten). Hinweis: Mirror, Type Features, Object = das Speichen-Extrude (oder Pattern), Plane = XY.

Bonus-Übung – Path Pattern entlang einer Kurve

  1. Skizze auf XY: einen Spline (mit Sketch → Create → Spline) zeichnen, vier Stützpunkte. Skizze schließen.
  2. Eine Kugel als Element: Create → Sphere, ⌀5, an einem der Spline-Endpunkte platzieren.
  3. Path Pattern. Create → Pattern → Path Pattern. Pattern Type: Bodies, Body = die Kugel. Path = der Spline. Quantity: 12. Distance Type = Spacing. OK. Die Kugeln verteilen sich gleichmäßig entlang der Kurve.
Erwartetes Ergebnis Ein Lüftungsgitter mit parametrischer Lochanzahl, ein Speichenrad mit kreisförmigem Pattern und gespiegelten Speichen, sowie eine Kette von Kugeln entlang eines Splines.

Freie Transferaufgabe – Lochwand mit Aussparung

Konstruiere eine 200×120×4 mm Lochwand mit ⌀5-Löchern im 12 mm-Raster. Lasse mittig eine 3×3-Lücke frei (für ein Beschriftungsschild).

  • Lochanzahl pro Richtung über Parameter floor.
  • Mittlere 3×3 Instanzen via Suppress ausschalten.
  • Bei Vergrößerung der Wand entstehen automatisch mehr Löcher, die mittlere Lücke bleibt.

Akzeptanzkriterium: Datei Lektion-2-4-Lochwand, beim Verbreitern auf 300 mm bleibt die Lücke 3×3 in der Mitte.

Selbstkontroll-Quiz

1. Welcher Pattern-Type bleibt am performantesten bei vielen Wiederholungen?
Pattern auf Features (also auf Operationen) ist meist günstiger als auf Bodies. Es muss nicht jedes Mal die ganze Body-Geometrie neu berechnet werden.
2. Was machst du, wenn ein Pattern an einer Stelle ein Loch zu viel produziert?
Im Pattern-Dialog den Bereich Suppress aufklappen, in der Vorschau die unerwünschte Instanz anklicken. Sie wird ausgeblendet, das Pattern bleibt parametrisch.
3. Welche Achse brauchst du für ein Circular Pattern um eine Bohrung mit schräger Lage?
Eine Construction Axis durch die Bohrung, am einfachsten via Axis Through Cylinder. Eine Origin-Achse passt nur, wenn die Bohrung achsparallel liegt.
4. Wann ist Mirror besser als Circular mit 2 Instanzen?
Wenn das Bauteil wirklich spiegelsymmetrisch ist, nicht nur 180°-rotationssymmetrisch. Bei Spiegeln werden links/rechts vertauscht, bei Rotation würden sie verdreht. Schraubgewinde z. B.: spiegeln dreht den Drehsinn um, rotieren nicht.
5. Was bewirkt das Spacing-Type Spacing im Path Pattern?
Die Instanzen werden gleichmäßig zwischen Anfang und Ende der Kurve verteilt – unabhängig davon, wie lang die Kurve genau ist. Distance dagegen setzt einen festen Abstand und füllt nur, was reinpasst.

Häufige Stolperer

  • „Pattern verzerrt" → Compute Option auf Identical stellen, statt Optimized. Sicherer bei komplexen Features.
  • „Mirror zerstört Geometrie" → Mirror-Plane schneidet das Original. Plane verschieben oder andere Symmetrieebene wählen.
  • „Distance/Spacing falsch" → in Spacing-Modus ist die Zahl der Abstand zwischen zwei Instanzen, in Distance-Modus die Gesamtstrecke. Zwei verschiedene Logiken.
