User-Parameter und Variablen

⏱ 40 Minuten 🎯 Vertiefung 🧮 Formeln und Einheiten

Lernziele

  • Ich lege User-Parameter mit Einheit, Formel und Kommentar an.
  • Ich verwende Parameter direkt in Skizzen-Maßen, Extrudes, Fillets und Patterns.
  • Ich baue Abhängigkeiten zwischen Parametern (ein Parameter rechnet aus einem anderen).
  • Ich nutze Funktionen wie max, min, floor, sqrt, cos, PI.
  • Ich kann Parameter umbenennen, ohne mein Modell zu zerstören.

Vorwissen-Check

Wo legst du User-Parameter an?
Toolbar Modify → Change Parameters. Im Dialog gibt es zwei Bereiche: Model Parameters (automatisch von jeder Operation angelegt) und User Parameters (deine eigenen Variablen, mit Plus-Icon hinzufügen).
Welche Einheiten sind in Fusion-Parametern erlaubt?
mm, cm, in (Länge), deg, rad (Winkel), und einheitenlose Werte (für Anzahlen, Faktoren). Gemischte Einheiten in einer Formel werden automatisch konvertiert, solange sie im selben „Domain" liegen.

Theorie kompakt

Was sind User-Parameter wirklich?

Ein User-Parameter ist eine benannte Variable mit Einheit, optional mit Formel und Kommentar. Statt in einer Skizzenbemaßung „18" zu tippen, schreibst du plattenstaerke. Wenn du später im Parameter-Dialog plattenstaerke von 18 auf 15 änderst, rechnen sich Skizze, Extrude, Pattern, Fillet – alles, was darauf verweist – neu.

Naming-Konvention

Halte dich strikt daran, sonst wird's chaotisch:

  • kleine Buchstaben, keine Umlaute, keine Sonderzeichen.
  • Wörter mit _ trennen: wand_aussen, schraube_m4_klemmweite.
  • Anfangen mit Kontext: box_breite ist klarer als breite_box, weil Sortierung im Dialog hilft.
  • Einheit immer mitsetzen: 5 mm, nicht nur 5 – auch wenn Fusion oft toleriert.

Formeln und Funktionen

Im Wert-Feld kannst du rechnen:

wand_innen      = 2 mm
wand_aussen     = wand_innen + 1 mm
gesamtbreite    = 80 mm
schraube_abstand = gesamtbreite / 2 - 10 mm
diagonale       = sqrt(gesamtbreite^2 + 60 mm^2)
loch_anzahl     = floor(gesamtbreite / 20 mm)
winkel          = 360 deg / loch_anzahl

Wichtige Funktionen: sqrt(x), abs(x), min(a, b), max(a, b), floor(x), ceil(x), cos(x), sin(x), tan(x), PI.

Abhängigkeiten zwischen Parametern

Ein Parameter darf andere referenzieren, solange keine Schleife entsteht. Daraus baust du Konstruktions-Hierarchien: ein Master-Maß bestimmt ein abhängiges Maß. Beispiel: 

tisch_breite     = 600 mm        // Master
schraube_abstand = tisch_breite / 4   // abhängig
tisch_tiefe      = tisch_breite * 0.6  // abhängig

Ändert sich tisch_breite, ziehen die anderen automatisch mit. Praktisch: ein einziger Parameter steuert die Proportionen.

Parameter umbenennen

Heißt ein Parameter unglücklich, kannst du ihn im Dialog mit Doppelklick umbenennen. Fusion ändert alle Verweise im Modell automatisch mit. Tabu: Parameter im fertigen Modell löschen, wenn andere darauf verweisen – das produziert Fehler.

Profi-Tipp Lege ganz oben in deiner Parameterliste einen Header mit Kommentaren an (z. B. // Geometrie, // Verbindungen, // Abhängig). Du kannst Kommentar-Zellen einfach mit Text füllen und 0 mm als Wert setzen. Das macht große Modelle deutlich lesbarer.

Geführte Übung – parametrische Schraubdose

Wir bauen eine kleine Dose mit Schraubdeckel-ähnlichen Bohrungen. Alle Maße über Parameter.

  1. Neues Design. Lektion-2-2-Schraubdose im Projekt Fusion-Kurs.
  2. Parameter-Dialog öffnen. Modify → Change Parameters. Im Reiter User Parameters auf das +-Icon und nacheinander folgende Parameter anlegen: 
    // — Geometrie —
dose_innen       = 60 mm    Innendurchmesser der Dose
dose_hoehe       = 40 mm    Gesamthöhe
wand             = 2 mm     Wandstärke
deckel_dicke     = 3 mm     Materialdicke des (späteren) Deckels

// — Verschraubung —
schraube_d       = 3 mm     Durchmesser für M3-Schraube (mit Spiel)
schraube_anzahl  = 4        einheitenlos
schraube_kreis_d = dose_innen + wand + 6 mm   Lochkreis-Durchmesser

// — Abgeleitet —
dose_aussen      = dose_innen + 2 * wand
flansch_d        = schraube_kreis_d + 2 * schraube_d
    OK.
    Hinweis Die Kommentare gibst du im Comment-Feld ein, nicht im Wert. Wenn Fusion eine Eingabe als Formel nicht akzeptiert, prüfe Einheiten (oben/unten konsistent: mm mit mm).
  3. Skizze auf XY. Kreis vom Origin mit C, Durchmesser bemaßen mit D: flansch_d. Skizze schließen.
  4. Flansch extrudieren. E, Profil = der Kreis. Distance = deckel_dicke, Operation = New Body. OK.
  5. Auf Oberseite Skizze für die Dose. Click auf die obere Fläche → Create Sketch. Kreis vom Origin, Durchmesser = dose_aussen. Skizze schließen.
  6. Dose nach oben extrudieren. E, Distance = dose_hoehe, Operation = Join. OK.
  7. Innen aushöhlen mit Shell. Modify → Shell. Wähle die Oberseite des Dosenkörpers. Inside Thickness = wand. OK.
  8. Schraubenkreis als Skizze. Click auf die obere Fläche des Flansches → Create Sketch. Kreis vom Origin, Durchmesser = schraube_kreis_d. Diesen Kreis mit X in eine Construction Line wandeln (gestrichelt). Auf dem Hilfskreis einen kleinen Kreis platzieren, Durchmesser = schraube_d. Skizze schließen.
  9. Bohrung extrudieren. E, Profil = kleiner Kreis. Distance = -deckel_dicke (negativ, durch den Flansch nach unten), Operation = Cut. OK. Wir patternen sie gleich.
  10. Circular Pattern (Vorgriff auf Lektion 4). Create → Pattern → Circular Pattern. Pattern Type = Features, das letzte Cut-Feature wählen, Achse = Z-Achse (Origin), Quantity = schraube_anzahl, Total Angle = 360°. OK.
  11. Stresstest der Parameter. Modify → Change Parameters. Ändere:
    • dose_innen auf 100 → die ganze Dose wird größer, Schraubenkreis wandert mit.
    • schraube_anzahl auf 6 → es sind plötzlich 6 Bohrungen.
    • wand auf 3 → Außendurchmesser wächst, Shell rechnet neu.
    Zurück zu den Originalwerten oder behalte deine Variante.
  12. Parameter umbenennen testen. Im Parameter-Dialog Doppelklick auf dose_hoehe, ändere zu dose_h. OK. Schau ins Modell – die Extrusion verweist jetzt auf den neuen Namen, ohne dass etwas kaputtging. Stell den Namen zurück oder lass ihn so.
  13. Speichern. Versionskommentar: „Voll parametrische Dose".
Erwartetes Ergebnis Eine Dose, die bei einer einzigen Parameteränderung sauber auf jede sinnvolle Größe skaliert. Anzahl der Bohrungen ist parametrisch, Wandstärke und Lochkreis ebenfalls.

Freie Transferaufgabe – Mutter mit M-Größe

Konstruiere eine generische Sechskantmutter ohne Gewinde mit folgenden Parametern: 

m_groesse  = 5 mm           // Nenndurchmesser
sw         = m_groesse * 1.6  // Schlüsselweite (Faustformel)
hoehe      = m_groesse * 0.8  // Mutternhöhe
loch       = m_groesse + 0.2 mm // Toleranz

Akzeptanzkriterium: Wenn du m_groesse auf 8 setzt, entsteht automatisch eine M8-Mutter mit korrekten Proportionen, ohne weitere Anpassung.

Selbstkontroll-Quiz

1. Was ist der Unterschied zwischen User Parameters und Model Parameters?
User Parameters legst du an und benennst sie sinnvoll. Model Parameters erzeugt Fusion automatisch für jede Operation (z. B. d12 für ein Maß). User Parameters sind die saubere Schnittstelle, die du auch in 6 Monaten noch verstehst.
2. Wie nutzt du den Wert eines Parameters in einem Skizzenmaß?
Statt einer Zahl im Bemaßungsdialog einfach den Parameternamen tippen, z. B. dose_innen. Auch Formeln sind erlaubt: dose_innen / 2.
3. Welche Einheiten sind in Parametern erlaubt?
Längenmaße (mm, cm, in), Winkel (deg, rad) und einheitenlose Werte für Anzahlen oder Faktoren. Mischen ist erlaubt, solange Fusion einheitlich rechnen kann.
4. Was passiert beim Umbenennen eines Parameters?
Fusion aktualisiert automatisch alle Verweise im Modell. Solange du im Dialog umbenennst (nicht löschst und neu anlegst), bleibt das Modell intakt.
5. Wann lohnt sich eine abgeleitete Variable?
Immer, wenn ein Wert mathematisch aus einem anderen folgt. dose_aussen = dose_innen + 2 * wand ist sicherer als zwei unabhängige Maße – sie können nicht inkonsistent werden.

Häufige Stolperer

  • „Parameter wird rot" → Schleife (A bezieht sich auf B, B auf A) oder undefinierter Name. Tippfehler prüfen.
  • „Wert kann nicht angewendet werden" → Einheiten passen nicht. Bei Anzahl unbedingt einheitenlos lassen, bei Längen mm sicherstellen.
  • „Parameter taucht nicht in der Liste auf" → versehentlich unter Favorites filtern. Zur Vollansicht oben den Filter zurücksetzen.
