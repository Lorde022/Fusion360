Export für CNC und Lasercutter

⏱ 50 Minuten 🎯 Holzwerken-Fokus ⚡ DXF · CAM

Lernziele

  • Ich exportiere 2D-Konturen als DXF aus Sketches und ebenen Flächen.
  • Ich nutze Layer/Farben in DXF, um Schneiden und Gravieren zu trennen.
  • Ich kenne die wichtigsten Schritte im Manufacture-Workspace (CAM): Setup, Stock, 2D Contour.
  • Ich erzeuge G-Code für eine einfache CNC-Tasche oder Außenkontur.
  • Ich kenne die Stolpersteine: Eckenradius, Aufmaß für Fräser-Kompensation, Zwiebel/Lead-In.

Vorwissen-Check

Was ist DXF?
Drawing Exchange Format. 2D-Vektorformat von Autodesk, Standard für Lasercutter und einfache CNC-Workflows. Enthält Linien, Bögen, Polygone, Layer.
Was ist G-Code?
Steuersprache für CNC-Maschinen. Befehle wie G1 X100 Y50 F500 bedeuten „lineare Bewegung mit Vorschub". Aus dem CAM-Workspace generiert Fusion G-Code, den deine Maschine versteht.

Theorie kompakt

Zwei Wege aus Fusion zur Fertigung

  • 2D-DXF-Export: für Lasercutter, Wasserstrahl und einfache CNCs, die mit eigener CAM-Software arbeiten (z. B. LightBurn, Estlcam).
  • G-Code aus Manufacture: für CNCs, bei denen du direkt die Toolpaths planst – Hobby-CNCs (Sienci, Shapeoko, OpenBuilds) und Profimaschinen.

DXF aus Sketch oder Fläche

Methode 1 – aus einer Skizze:

  1. Skizze im Browser auswählen.
  2. Rechtsklick → Save as DXF.
  3. Speicherort, Datei.

Methode 2 – aus einer ebenen Fläche eines Bauteils:

  1. Auf die Fläche klicken (oder Rechtsklick).
  2. Rechtsklick → Create Sketch auf dieser Fläche, dann Project der Außenkontur.
  3. Skizze schließen, dann Save as DXF.

Fräser-Kompensation

DXF zeigt nur die Außenkonturen. Beim Schneiden mit Lasern oder Fräsern entsteht ein Schnittfugen-Versatz: ein 3-mm-Fräser entfernt 3 mm Material entlang der Kontur. Die fertige Bauteilkante liegt also je nach Schnittseite (innen/außen) versetzt.

  • Beim CNC-Schneiden: in der CAM-Software Tool Compensation (links/rechts der Linie) einstellen.
  • Beim Lasercutter: Schnittfugen-Versatz (etwa 0,1–0,2 mm bei 6-mm-Multiplex) im Slicer berücksichtigen oder über Kerf-Compensation einstellen.

Layer / Farben in DXF

LightBurn und Estlcam unterscheiden Operationen über Linienfarbe oder Layer. In Fusion legst du das vor dem Export so an:

  • Schneidkonturen in einer Skizze.
  • Gravur-Geometrie in einer separaten Skizze.
  • Beide DXF-Dateien getrennt exportieren – oder beide in einer Sketch und Linien als Construction Lines markieren (werden in DXF nicht exportiert) bzw. mit verschiedenen Farben über die Sketch Palette bemalen.

Manufacture-Workspace im Schnellüberblick

  1. Setup – Werkstückkoordinaten und Materialgröße (Stock) festlegen.
  2. 2D / 3D Toolpaths – Operationen wählen (2D Contour, 2D Pocket, Drill, Adaptive Clearing).
  3. Tool – Werkzeug aus Library wählen oder selbst anlegen.
  4. Simulate – Vorschau der Werkzeugbewegung.
  5. Post Process – G-Code in passendem Dialekt (Grbl, Mach3, LinuxCNC, …) ausgeben.
Test mit Schaumstoff/MDF Erste CNC-Tests nie im teuren Multiplex. Reststücke MDF oder Hartschaum sind perfekt für Trockenläufe. Manche Profis machen Air Cuts: Werkstück gar nicht einspannen, Spindel läuft trocken zur Sichtkontrolle.

Geführte Übung – DXF einer Plattenkomponente

Wir nehmen ein Bauteil aus der Box-Übung in 4.2 und exportieren es als DXF.

  1. Box-Datei öffnen. Lektion-4-2-Box oder eine vergleichbare Box.
  2. Eine Seitenwand isolieren. Im Browser nur die linke Seitenwand sichtbar lassen, alle anderen ausblenden.
  3. Skizze auf Außenfläche. Klicke auf die Außenfläche der Seitenwand → Create Sketch. Sketch → Create → Project: Außenkontur projizieren. Auch alle Innenausschnitte (Zinken, Bohrungen) projizieren – sie werden in der DXF mitgenommen. Skizze schließen.
  4. Skizze als DXF speichern. Im Browser unter Sketches die neue Skizze rechtsklicken → Save as DXF. Speicherort, Dateiname Seitenwand-links.dxf.
  5. DXF prüfen. Datei in einem DXF-Viewer öffnen (LightBurn, QCAD, Inkscape). Du siehst die Außenkontur und alle Ausschnitte. Maßeinheit prüfen: Fusion exportiert Millimeter, manche Programme nehmen Inch an.
  6. Wiederholen für alle Bauteile. Sechs Plattenteile = sechs DXF-Dateien (oder ein „Nesting"-DXF mit allen Teilen nebeneinander – manuell durch Aneinanderlegen der Ansichten in der CAM-Software).

Bonus – CAM-Crashkurs für die Außenkontur

  1. Workspace wechseln. Oben links auf Design → Manufacture.
  2. Setup anlegen. Toolbar: Setup.
    • Operation Type: Milling.
    • Work Coordinate System (WCS): an einer Bauteilecke setzen, z. B. linke obere Ecke der Seitenwand.
    • Stock: Definition automatisch Relative size box oder manuell, z. B. 200×100×18 mm Multiplex.
    OK.
  3. 2D Contour. Toolbar 2D → 2D Contour.
    • Tool: einen 6-mm-Schaftfräser anlegen (Library → New Tool).
    • Geometry: die Außenkontur des Bauteils.
    • Heights: Top of Stock = obere Fläche, Bottom = untere Fläche.
    • Passes: Stepdown 3 mm (also drei Durchgänge bei 18 mm Material).
    OK. Toolpath wird berechnet.
  4. Simulate. Toolpath rechtsklicken → Simulate. Du siehst, wie der Fräser das Bauteil austrennt.
  5. Post Process. Toolpath rechtsklicken → Post Process. Wähle den passenden Post (z. B. grbl für Hobby-CNC). G-Code-Datei wird gespeichert.
Erwartetes Ergebnis Du hast für ein einzelnes Bauteil sowohl eine DXF (Lasercutter / Estlcam-tauglich) als auch eine G-Code-Datei (CAM-Workflow). Beide Wege sind dir vom Ablauf vertraut.

Freie Transferaufgabe – Nesting

Erzeuge ein Nesting (mehrere Bauteile auf einer Platte) für die Box-Bauteile:

  • Eine neue Skizze auf XY anlegen.
  • Mit Project die Außenkonturen der einzelnen Bauteile aus dem Modell in diese Skizze projizieren.
  • Per Move die Konturen so verschieben, dass sie auf einer 1200×600 mm-Platte unterzubringen sind, mit mindestens 5 mm Abstand zwischen Bauteilen.
  • Skizze als DXF exportieren.

Akzeptanzkriterium: Eine DXF-Datei Box-Nesting.dxf, die du als kompletten Schnittplan an einen Lasercutter oder eine CNC-Fräse geben könntest.

Selbstkontroll-Quiz

1. In welchem Format gibst du Konturen an einen Lasercutter?
DXF. Manche Lasercutter akzeptieren auch SVG, das funktioniert ebenso.
2. Was bedeutet Tool Compensation?
Versatz der Werkzeugmittellinie zur gewünschten Bauteilkante – links für innen, rechts für außen, oder „On" für mittig. So ist die fertige Kante exakt dort, wo sie sein soll.
3. Was ist der erste Schritt im Manufacture-Workspace?
Ein Setup anlegen – darin definierst du Werkstück-Koordinaten und Stock (Materialgröße/Form). Ohne Setup keine Toolpaths.
4. Was passiert in der Simulation?
Fusion zeigt visuell, wie der Fräser am Werkstück arbeitet. Du erkennst Kollisionen, falsche Eintauchtiefen oder schlechte Bearbeitungsstrategien, bevor die Maschine läuft.
5. Welcher Post Processor passt für Hobby-CNCs?
Für Grbl-basierte Maschinen (Shapeoko, Onefinity, Workbee usw.) der grbl-Post. Andere Maschinen (Mach3/4, LinuxCNC, Acorn, Centroid) haben jeweils eigene Posts in der Fusion-Library.

Häufige Stolperer

  • DXF in falscher Maßeinheit → Empfänger-Software auf mm umstellen, oder beim Export prüfen.
  • Innenecken zu spitz für CNC → Eckenradius im CAD setzen, mindestens Fräserradius.
  • G-Code passt nicht → falscher Post Processor. Hersteller-Doku der CNC verrät, welcher Dialekt nötig ist.
